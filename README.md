## Handwritten Alphabet Recognition (A–Z)
# Overview
A deep learning project that recognizes handwritten English alphabets (A–Z) using a CNN model and Streamlit web app.

# Model
CNN (Convolutional Neural Network)
Trained on A–Z handwritten dataset (28×28 images)
26 output classes (A–Z)
Accuracy: ~99%
# Tech Stack
Python, TensorFlow/Keras, NumPy, Pillow, Streamlit

# Features
Upload handwritten image
Predict alphabet (A–Z)
Show confidence score
